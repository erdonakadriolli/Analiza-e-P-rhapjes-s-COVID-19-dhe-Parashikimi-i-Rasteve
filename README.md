# Analiza-e-P-rhapjes-s-COVID-19-dhe-Parashikimi-i-Rasteve
Lënda: Shkenca e të Dhënave dhe Vizualizimi me Python. Ekipa: Erdona Kadriolli, Fatlum Syla, Yll Bytyqi. Ky projekt analizon dhe vizualizon përhapjen e COVID-19 duke përdorur të dhëna nga Our World in Data, me fokus në Kosovë dhe rajon, përmes grafikëve, heatmap dhe një modeli të thjeshtë parashikimi.
