# COVID-19 Data Analysis and Visualization

## 📊 Project Description
This project analyzes COVID-19 data using Python. It includes data cleaning, analysis, visualization, and prediction of cases.

## 📁 Dataset
Data is taken from Our World in Data.

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## 📈 Features
- Data cleaning and preprocessing
- Statistical analysis
- Data visualization (charts, heatmaps)
- Prediction of future cases
- Interactive dashboard
